Sobre mim

Comecei na área de desenvolvimento web voltado ao front-end à 8 meses no momento em que trabalhava na Construtora LDN LTDA. Comecei como estagiário de Engenharia Civil na empresa sendo um dos responsáveis pelo sistema de gestão e qualidade da construtora, após um tempo na área de trabalho, comecei a aplicar linguagens de programação nos serviços em que executava a fins de praticidade e prazo de entrega. Com o tempo, passei a me aprofundar mais na área de Desenvolvedor entrando em novas linguagens e me aprimorando nas que eu ja tinha conhecimento. Hoje procuro um novo trabalho, onde posso evoluir ainda mais na área de desenvolvimento, além de tentar ao máximo entregar um bom desempenho para qualquer empresa em que ingressar.

HTML

O HTML5 foi criado com o objetivo de permitir que os desenvolvedores criem sites mais complexos e interativos, com elementos multimídia, gráficos avançados, animações e outras funcionalidades sem precisar recorrer a plug-ins de terceiros, como o Adobe Flash

CSS

CSS (Cascading Style Sheets) é uma linguagem de folhas de estilo utilizada para definir a aparência e o layout de páginas da web.

JAVASCRIPT

JavaScript é uma linguagem de programação de alto nível, interpretada, orientada a objetos e amplamente utilizada na programação de aplicações para a web.

Desenvolvimento Web

Desenvolvimento web é o processo de criação de sites, aplicações e outros produtos para a web, que podem ser acessados por meio de navegadores da web.

Desenvolvimento Mobile

Desenvolvimento mobile é o processo de criação de aplicativos móveis para smartphones, tablets e outros dispositivos móveis.

Uso de API

API (Application Programming Interface) é um conjunto de regras e protocolos que especificam como diferentes componentes de software devem interagir entre si. Em outras palavras, uma API define uma interface para permitir que diferentes aplicativos, sistemas e serviços se comuniquem e compartilhem informações.
